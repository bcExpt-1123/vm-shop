# Cezerin - Ecommerce Progressive Web Apps

Cezerin2 is React and Node.js based eCommerce platform. Allows creating a Progressive Web Apps. This is based from cezerin after project went dead. 

Built with:
* Node.js v8.9
* React v16
* Redux
* Express
* Babel
* WebPack 4
* MongoDB

## Installation

### Requirements
* Node.js >= 8
* MongoDB >= 3.2
